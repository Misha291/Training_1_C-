# Калькулятор скидки (C#)

Консольная программа для расчёта цены товара со скидкой.

## Как работает
1. Ввод цены товара
2. Ввод процента скидки
3. Вывод итоговой цены с 2 знаками после запятой

## Пример
```
Введите цену товара: 1000
Введите скидку в процентах: 15
Итоговая цена: 850.00 руб.
```

## Формула
`итоговая_цена = цена × (1 − скидка / 100)`

## Запуск
Скомпилируйте и запустите через Visual Studio (Ctrl+F5).
